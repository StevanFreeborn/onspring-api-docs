# Apps {% #apps %}

These are objects that represent apps and/or surveys in an Onspring instance. You can retrieve a list of apps that the given [API Key](#authentication) has access to.

## App Properties

{% table %}

- Property Name
- Data Type
- Description

---

- Id
- `number`
- The id of the app.

---

- Name
- `string`
- The name of the app.

{% /table %}
