# Apps {% #apps %}

These are objects that represent apps and/or surveys in an Onspring instance. You can retrieve a list of apps or a specific app by its Id.

## App Properties

{% table %}

- Property Name
- Data Type
- Description

---

- href
- `string`
- The URL of the app.

---

- id
- `number`
- The id of the app.

---

- name
- `string`
- The name of the app.

{% /table %}
