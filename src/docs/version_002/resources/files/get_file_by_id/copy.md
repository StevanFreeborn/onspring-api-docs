# Get File by Id {% #get-file-by-id %}

This endpoint returns the file with the given id.

## Path Parameters

{% table %}

- Parameter Name
- Data Type
- Description

---

- recordId
- `number`
- The id of the record that contains the file.

---

- fieldId
- `number`
- The id of the field that contains the file.

---

- fileId
- `number`
- The id of the file.

{% /table %}
